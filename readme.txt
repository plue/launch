
Launch
======

Am Anfang jeden Erfolgs steht eine inspirierende Idee.  
Ihre Kommunikation macht sie auch für andere erlebbar.  

Ganz einfach. Ganz umsonst.  

https://launch.plue.me  


Anleitung
---------

1. Lade die Website von https://launch.plue.me

2. Öffne index.html im Browser, um die Website anzusehen

3. Öffne index.html in einem Texteditor, um die Website zu bearbeiten (Empfehlung: https://www.sublimetext.com)

4. Lade index.html auf deine Domain, um die Website zu veröffentlichen (Empfehlung: https://guide.plue.me/04-domains)
